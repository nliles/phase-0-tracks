Check out this stylish travel site: MeltingButter. What makes this site good? The website is simple but has an elegant style. It isn't too cluttered and is easy to navigate around in. I especially like the toolbar on the top of the page. It is attractive and presents all of the contents right away. While there are a lot of pictures on the home page, it does not appear to be too busy. I would like to learn to create a toolbar like this for my page. Do you recognize any HTML elements? The site contains many basic HTML elements that include some attributes I haven't used yet.
